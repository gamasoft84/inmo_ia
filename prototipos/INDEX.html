Cómo usar estos archivos en VS Code + Claude Code:
1. Abre este archivo en Live Server (extensión de VS Code) o en tu navegador
2. Navega a la fase que estás construyendo
3. Toma screenshot de la pantalla que necesitas (Cmd+Shift+4 en Mac)
4. Adjunta el screenshot en Claude Code: "Construye esta pantalla exactamente"
5. Claude Code replica el diseño con el Design System de InmoIA
